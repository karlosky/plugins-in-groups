Organize the WP plugins in the groups. 
With this plugin you can keep your plugins page clear, manage them in bulk and filter plugins by the tags.

== Description ==

With this plugin you can create the groups on the plugins page.
Then you can assign the plugins to these groups and filter them by the tags.
Each plugin can have more tags (groups) assigned so you can sort and filter them as you want. It allows you to keep your plugins page clean and you can manage the plugins in bulk.


== Installation ==

1. Install the plugin from the WordPress Plugins Installer
2. Activate the plugin
3. Manage and filter the plugins by the groups on the 'Plugins' page

== Frequently Asked Questions ==

= How it should be configured? =

It doesn't have to! You should just install and activate the plugin and you can create the new groups of the plugins.

= Does it work on the multisite WP? =

Yes! You can create the groups for all the website in your WP network.
